Help Center & Support

Need assistance using the TruthLens Intelligence platform? Reference the guides below to troubleshoot common issues.

"Could not extract meaningful text" Error

This occurs when our scraper is blocked by an aggressive anti-bot firewall on the target website rendering the URL analysis useless.

Solution: Please manually copy the text directly from the news article and paste it into the "Global" pure-text scanning tab.

How to access the Admin Console?

The Admin Console is restricted strictly to system oversight personnel. You can access it via the left sidebar. If you need to approve disputes, initiate retraining, or view telemetry, ensure your session is authenticated via DB Admin credentials.

Platform Requirements

TruthLens is highly optimized for modern web browsers. Ensure JavaScript is enabled for real-time confidence metric renderings and dashboard interactive transitions.
